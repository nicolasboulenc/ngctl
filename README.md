## Intro

`nsm` allows you to quickly manage sites on NGINX via the command line.

**Example:**
```sh
$ nsm start
location enabled: /home/nicolas/dev/nsm/
 --> http://localhost:8080/
$ nsm version
v16.9.1
```

Simple as that!
