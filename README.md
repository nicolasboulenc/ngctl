## Intro

`ngctl` allows you to quickly manage sites on NGINX via the command line.

**Example:**
```sh
$ ngctl start
location enabled: /home/nicolas/dev/ngctl/
 --> http://localhost:8080/
$ ngctl version
v16.9.1
```
